# Project Aegis
1.打造一套專門給遊戲、社群平台、聊天室使用的 即時安全治理系統。
2.核心目標：讓騷擾者、誘導者、慣犯無法輕易重返社群，保護正常玩家。
3.像防毒軟體保護電腦一樣，Aegis 保護線上社群。

# Project Aegis

AI moderation system designed to protect online games and communities.

## Features

- Real-time harassment detection
- Grooming risk alerts
- VPN evasion scoring
- Community justice review system
- Fair punishment engine
- Admin dashboard

## Mission

Safer digital worlds for everyone.
